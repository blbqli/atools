# ATools Chrome Extension (sidepanel)

This folder is a standalone extension project (isolated dependencies).

## Dev

1. `cd extension && npm install`
2. `npm run dev` (build watch → outputs to `extension/dist/`)
3. Chrome → `chrome://extensions` → Enable Developer Mode → **Load unpacked** → select `extension/dist`
4. After code changes, click **Reload** on the extension.

## Build

- `cd extension && npm run build`

## Manifest

- Source of truth: `extension/src/manifest.json`
- Generated on build: `extension/dist/manifest.json` (do not edit manually)

## Tools meta

- Generated by repo root `npm run generate:manifests`: `extension/tools-meta*.json`
- Copied on extension build: `extension/dist/tools-meta*.json`
